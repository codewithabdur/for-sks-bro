# SeniorSolutionsApp

Easiest way to manage your health, medicines, emergencies all in one place

Live on -   https://sachinkrsundram.github.io/SeniorSolutionsApp/


#Features

Medicine Reminder
Blood Test Remainder
Medical History
Senior Social Media Plateform
Virtual Assistant


# Technologies Used

React.js
Google Assitant API
